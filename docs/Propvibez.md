# **App Name**: Shaily Madaan

## Core Features:

- About Shaily Section: A section detailing Shaily Madaan's background and experience as Founder & Director of ProipVibez.
- YouTube Portfolio: Display a curated list of YouTube videos, showcased within a carousel.
- Social Media Showcase: An overview of ProipVibez's presence across other social media platforms, each represented with its logo.
- Enquiry Form: A contact form for inquiries with fields for name, email, subject, and message.

## Style Guidelines:

- Primary color: Deep purple (#673AB7) for creativity and leadership.
- Background color: Light lavender (#EDE7F6), a soft tint of purple, creates a calm, professional ambiance.
- Accent color: Blue-violet (#3F51B5) for highlights and call-to-action buttons.
- Font: 'Inter', a grotesque sans-serif, for both headlines and body text, lending a modern and approachable feel to the site. Note: currently only Google Fonts are supported.
- Single-page layout with clearly defined sections for About, Portfolio, Social Media, and Contact.
- Clean, professional icons for social media links.
- Subtle transition animations as the user scrolls through sections.
